# 🎯 AI 学习大纲

> 从 AI 小白到 LLM 应用开发者的完整学习路径

## 📋 学习路线总览

```
机器学习基础 (2-3周) → 神经网络 (2-3周) → 大语言模型 (3-4周) → Agent开发 (2-3周)
```

**学习定位**：专注于理解原理和应用开发，不深入基础设施(Infra)层面

---

## 第一阶段：机器学习基础 (2-3周)

> **主要教材**：《机器学习》周志华（西瓜书）
> **配套视频**：[B站讲解 BV1gG411f7zX](https://www.bilibili.com/video/BV1gG411f7zX/)

### 🎯 学习目标
- 理解机器学习的核心思想：从数据中学习规律
- 掌握监督学习、无监督学习的基本概念
- 理解模型训练、验证、测试的完整流程
- 建立对"学习"和"泛化"的直觉认知

### 📚 核心概念

#### Week 1：绪论（第1章）📍 [详细学习计划](notes/01-机器学习基础/Week1-学习计划.md)
- [ ] **机器学习定义**：从经验中学习，提升任务性能（T-E-P）
- [ ] **基本术语**
  - 样本、属性、标记、特征向量
  - 训练集、测试集、泛化能力
- [ ] **假设空间与版本空间**
- [ ] **归纳偏好**
  - 奥卡姆剃刀原则
  - NFL 定理（没有免费午餐）

#### Week 2：模型评估与选择（第2章）
- [ ] **经验误差与过拟合**
- [ ] **评估方法**：留出法、交叉验证、自助法
- [ ] **性能度量**：准确率、精确率、召回率、F1、ROC、AUC
- [ ] **比较检验与偏差-方差分解**

#### Week 3：线性模型 + 决策树（第3-4章）
- [ ] **线性回归与对数几率回归**
- [ ] **决策树基本算法**
- [ ] **划分选择**：信息增益、增益率、基尼指数
- [ ] **剪枝处理**

### 💡 学习要点
1. 以西瓜书为主，配套视频辅助理解
2. 重点理解概念，公式推导可参考南瓜书
3. 通过西瓜例子建立直觉认识

## 第二阶段：神经网络 (2-3周)

### 🎯 学习目标
- 理解神经网络的基本结构
- 掌握深度学习的核心概念
- 理解为什么深度学习能够处理复杂问题
- 为理解大语言模型打下基础

### 📚 核心概念

#### Week 1：神经网络基础
- [ ] **神经元模型**
  - 输入 → 加权求和 → 激活函数 → 输出
  - 激活函数的作用：引入非线性
- [ ] **多层感知机(MLP)**
  - 输入层、隐藏层、输出层
  - 前向传播：数据如何流经网络
- [ ] **反向传播**
  - 核心思想：从输出到输入逐层计算梯度
  - 理解直觉：误差如何指导参数更新

#### Week 2：深度学习关键组件
- [ ] **常见网络结构**
  - CNN（卷积神经网络）：处理图像的利器
  - RNN（循环神经网络）：处理序列数据
  - 理解它们为什么适合各自的任务
- [ ] **训练技巧**
  - Batch、Epoch、Learning Rate
  - Dropout、Batch Normalization（概念理解）
- [ ] **深度学习框架**
  - PyTorch 或 TensorFlow 基础使用

#### Week 3：通向语言模型
- [ ] **词嵌入(Word Embedding)**
  - 为什么需要将文字转换为向量
  - Word2Vec 的核心思想
- [ ] **序列建模挑战**
  - RNN 的局限性：长程依赖问题
  - 引出 Attention 机制的必要性

### 💡 学习要点
1. 理解神经网络是"可学习的函数"
2. 重点掌握网络结构的直觉
3. 为后续 Transformer 学习做铺垫

## 第三阶段：大语言模型 (3-4周) ⭐ 重点

### 🎯 学习目标
- 深入理解 Transformer 架构
- 掌握 LLM 的工作原理
- 理解 Prompt Engineering 的本质
- 学会使用各种 LLM API

### 📚 核心概念

#### Week 1：Transformer 架构
- [ ] **Attention 机制**
  - 自注意力(Self-Attention)的直觉
  - Query、Key、Value 的含义
  - 为什么 Attention 能捕获长程依赖
- [ ] **Transformer 结构**
  - Encoder-Decoder 架构
  - Multi-Head Attention
  - Position Encoding：为什么需要位置信息
  - Feed Forward Network
- [ ] **关键论文理解**
  - 《Attention Is All You Need》核心思想

#### Week 2：大语言模型原理
- [ ] **预训练与微调**
  - 预训练任务：下一个词预测(Next Token Prediction)
  - 为什么"预测下一个词"能学到语言知识
  - 微调(Fine-tuning)的意义
- [ ] **GPT 系列演进**
  - GPT-1 → GPT-2 → GPT-3 → GPT-4
  - 涌现能力(Emergent Abilities)
  - 规模定律(Scaling Laws)
- [ ] **RLHF（人类反馈强化学习）**
  - 为什么需要 RLHF
  - SFT → Reward Model → PPO 流程
  - InstructGPT / ChatGPT 的训练过程

#### Week 3：Prompt Engineering
- [ ] **Prompt 的本质**
  - 上下文学习(In-Context Learning)
  - Few-shot、Zero-shot、Chain-of-Thought
- [ ] **Prompt 设计技巧**
  - 角色设定、任务描述、格式约束
  - 分步推理、示例引导
  - 常见 Prompt 模式和最佳实践
- [ ] **高级技巧**
  - 自我一致性(Self-Consistency)
  - 思维树(Tree of Thoughts)
  - 检索增强生成(RAG)基础概念

#### Week 4：LLM 应用实践
- [ ] **API 使用**
  - OpenAI API 完整使用
  - 其他模型 API（Claude、文心一言等）
  - Token、Temperature、Top-p 等参数理解
- [ ] **常见应用模式**
  - 文本生成、摘要、翻译
  - 代码生成、问答系统
  - 结构化输出(JSON Mode)
- [ ] **局限性与解决方案**
  - 幻觉(Hallucination)问题
  - 上下文长度限制
  - 知识截止日期

### 💡 学习要点
1. **Transformer 是重中之重**，花足够时间理解
2. 理解"下一个词预测"为什么如此强大
3. Prompt Engineering 是与 LLM 交互的核心技能

## 第四阶段：Agent 开发 (2-3周)

### 🎯 学习目标
- 理解 AI Agent 的概念和架构
- 掌握 Agent 开发的核心框架
- 能够独立开发简单的 Agent 应用
- 了解多 Agent 协作系统

### 📚 核心概念

#### Week 1：Agent 基础
- [ ] **什么是 AI Agent**
  - 感知 → 思考 → 行动 循环
  - Agent = LLM + 记忆 + 工具 + 规划
- [ ] **Agent 核心组件**
  - 规划(Planning)：任务分解、反思
  - 记忆(Memory)：短期/长期记忆
  - 工具使用(Tool Use)：扩展能力边界
- [ ] **ReAct 模式**
  - Reasoning + Acting
  - 思考-行动-观察循环

#### Week 2：Agent 开发实践
- [ ] **LangChain 框架**
  - Chain、Agent、Tool 概念
  - 常用组件和使用方法
  - 构建简单的对话 Agent
- [ ] **Function Calling**
  - OpenAI Function Calling 机制
  - 工具定义和调用流程
  - 自定义工具开发
- [ ] **RAG 深入**
  - 向量数据库选择
  - 文档切分策略
  - 检索优化技巧

#### Week 3：高级 Agent 开发
- [ ] **多 Agent 系统**
  - Agent 间通信和协作
  - 任务分配和协调
  - AutoGen、CrewAI 等框架
- [ ] **Agent 设计模式**
  - 规划器-执行器模式
  - 监督者模式
  - 辩论/讨论模式
- [ ] **实战项目**
  - 构建一个完整的 Agent 应用
  - 从需求分析到部署上线

### 💡 学习要点
1. Agent 是 LLM 能力的放大器
2. 重点掌握工具使用和 RAG
3. 通过实践项目巩固所学
