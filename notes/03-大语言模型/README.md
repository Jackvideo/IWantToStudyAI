## 💡 关键概念卡片

| 概念 | 一句话解释 | 掌握程度 |
|------|-----------|----------|
| Self-Attention | 让每个词能"看到"序列中所有其他词 | ⬜ |
| Q/K/V | Query查询、Key键、Value值，Attention三要素 | ⬜ |
| 位置编码 | 告诉模型每个词在句子中的位置 | ⬜ |
| 下一词预测 | GPT 的核心训练目标 | ⬜ |
| RLHF | 用人类反馈来对齐模型行为 | ⬜ |
| In-Context Learning | 通过上下文示例来引导模型 | ⬜ |
| CoT | 引导模型分步推理 | ⬜ |
| Temperature | 控制输出随机性的参数 | ⬜ |
| 幻觉 | 模型编造不存在的信息 | ⬜ |
