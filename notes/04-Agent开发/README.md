## 💡 关键概念卡片

| 概念 | 一句话解释 | 掌握程度 |
|------|-----------|----------|
| Agent | LLM + 工具 + 记忆 + 规划 | ⬜ |
| ReAct | 推理-行动-观察循环 | ⬜ |
| Tool Use | 让 LLM 调用外部工具扩展能力 | ⬜ |
| RAG | 检索增强生成，开卷考试模式 | ⬜ |
| 向量检索 | 基于语义相似度的检索方式 | ⬜ |
| Function Calling | LLM 调用函数的标准化方式 | ⬜ |
| 短期记忆 | 对话上下文 | ⬜ |
| 长期记忆 | 持久化存储的知识 | ⬜ |
