# 🤖 我要学 AI

> 从零开始的 AI 学习之旅：机器学习 → 神经网络 → 大模型 → Agent

这是一个个人 AI 学习项目，记录从 AI 小白到掌握 LLM 应用开发的学习历程。

**学习定位**：专注于理解原理和应用开发，不深入基础设施(Infra)层面

## 🎯 学习目标

- [ ] 理解机器学习的核心概念（从数据中学习）
- [ ] 掌握神经网络的基本原理（深度学习基础）
- [ ] 深入理解大语言模型的工作机制（Transformer、GPT）
- [ ] 熟练掌握 Prompt Engineering
- [ ] 学会开发 AI Agent 应用（LangChain、RAG）

## 💡 学习建议

1. **不要追求面面俱到**，优先建立核心概念的直觉理解
2. **多用 AI 导师**，遇到不懂的概念就问
3. **勤做笔记**，用自己的话复述概念（费曼学习法）
4. **动手实践**，代码跑起来比只看文档更有效
5. **允许不完美**，学习是渐进的过程
